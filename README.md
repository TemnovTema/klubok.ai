# Клубок.ai — продуктовый лендинг

Концепт лендинга сервиса автономных AI-агентов. Отдельный Vite + React + TypeScript проект.

```bash
npm install
npm run dev
```

Визуальное направление основано на презентации Kodo.media: чёрный фон, бирюзовый акцент, крупная типографика. Продуктовые экраны — самостоятельные концепты рабочего центра, настройки агента и мобильного подтверждения действий.
